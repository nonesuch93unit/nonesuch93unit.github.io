    • 第一个系列技术文章，我来讲讲如何从零开始创建自己的博客，这里的从零开始，是指没有任何编程基础，跟着本教程做下去，你将拥有属于自己的博客。
    注意，本系列文章主要针对windows操作系统 + hexo3.x。

    为什么写这个系列？

    原因有三：

    1. 因为自己在看别人的教程的时候发现了一些别人没有提及的问题本系列会介绍我遇到问题时的解决办法，给大家一个尝试的方案。
    2. 网上的教程多是一篇长篇大论，令人望而生畏。因此本系列分成了几篇，前四篇的内容讲解如何搭建自己网站，最后一篇讲解如何个性化自己的网站。
    3. 网上一些资料的版本较旧，关键地方的图片也已经过时，容易引起误解。

    总论

    总的来说，搭建自己的博客比较简单，但是也比较折腾。主要原因是其中的步骤较多，难免会在某一步遇到问题。我会尽量清晰详细的讲解每一步过程，只要你一步步做，一定能拥有自己的博客。
    如果你在做的过程中遇到我没提到的问题，或者有什么更好的想法，欢迎随时与我联系（邮箱在博客下方）。

    构建博客的主要流程

    建立博客的主要流程为： 创建空间(github pages) -> 利用框架构建网站结构(hexo) -> 上传网页(hexo) -> 申请域名(godaddy) -> 建立DNS连接(DNSpod) -> 写博客(markdown)
    个性化的主要流程为：克隆jacman主题(hexo) -> 更改参数个性化自己的页面(jacman) 接下来挨个介绍：

    1. 创建空间：你网页的文字、图片需要存储在一个网上空间里，虽然也可以留在你电脑里，可是你的电脑不可能24小时连着网，总不能电脑断网了博客就不能访问了吧？网上空间我们用github pages，安全可靠。（事实上github托管了你的网站项目，在此不多做讨论）
    2. 构建框架：每一个站点都有不少页面，自然需要一个框架来管理。我们用现成的hexo来做这件事，简单快捷。
    3. 上传网页：之前已经提过，要将博文上传到网络空间里。
    4. 申请域名: 这个容易理解，上过网的都知道。比如你想用百度，就输入www.baidu.com，这就是百度的域名。同样的，自己的博客，也要申请一个名字。我们选择国外的godaddy网站申请，因为国内申请域名要存根，不太方便。godaddy支持支付宝，还是相当好用的。
    5. 建立DNS连接：如何将自己的网上空间和域名联系起来呢？自然是用DNS连接啦。(关于DNS的具体含义还请自行搜索)
    6. 写博客：到了这一步，你的网站已经建好了，终于可以写博客了。可惜写博客不能像office办公软件那样编辑，而要用到markdown语言，后面会介绍（它非常简单！）。
    7. 克隆主题：你可能不喜欢hexo自带的主题，这时我们可以用hexo克隆一些别人的主题下来。
    8. 个性化：克隆主题后进一步个性化，让它成为独一无二的博客！

    安装

    依次下载安装：1. Node.js 2. Git
    下载链接在主页上很容易找到，注意安装与系统兼容的版本。
