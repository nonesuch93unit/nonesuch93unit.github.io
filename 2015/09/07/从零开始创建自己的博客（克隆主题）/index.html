通过前面的几篇，我们已经创建出了自己的博客，但是我们希望在原来的基础上加一些自己的设计，使它更加美观。hexo支持很多主题模式，用户可以很简单地克隆主题，再加上自己设置的一些参数，使它变成自己独一无二的博客。
本文以克隆jacman为例。jacman以全扁平化风格设计，简洁美观，也是现在比较潮流的设计模式。感谢wuchong将他如此美观的设计开源！

克隆jacman主题

  1. 在站点文件夹根目录右键git bash，输入

  2. 打开站点根目录下的_config.yml文件（可用notepad++打开），将theme属性的值改为jacman。

  3. jacman安装好了，可以在根目录下hexo g然后hexo s查看！

个性化设置

根目录下的_config.yml

打开根目录下的_config.yml文件，里面有一些参数可以修改，下面我介绍几个常用的。

  1. title和subtitle：主标题和副标题
  2. author：作者
  3. Directory下面有一个列表，写着archive_dir: archives等项。他们是你的网站最上方的菜单项，如果你要加菜单项，要在这里写一句（举例，我要加入tags项，就在这写tag_dir: tags），然后在根目录\source文件夹中新建一个叫tags的文件夹，在里面建一个叫index.md的文件。
  4. per_page：每页显示的文章数

根目录\themes\jacman下的_config.yml

这个文件可以修改的参数更多：

  1. 菜单 menu：依然是网站最上的菜单栏，默认没有启用 /tags 和 /categories页面，如果需要启用请在博客目录下的source文件夹中分别建立tags 和 categories文件夹每个文件夹中分别包含一个index.md文件。内容为：

    layout: tags (或categories)

    title: tags (或categories)

  2. 控件 widgets: 它们就是我的网站旁边的这些分类、标签云、和友情链接。如何使用标签呢？在你写文章的时候在最上面这样写

    title: （五）从零开始创建自己的博客（克隆主题 + 个性化 + 杂项 + 总结）(Github Pages + Hexo + jacman)
    date: 2015-09-07 17:38:59
    tags: jacman
    标签中就会自动创建一个jacman标签出来，并且把这篇文章包括在这个标签中了。

  3. 图片相关Image：本主题可以设置网站相关图片，例如网站图标（favicon）、网站logo（imglogo）、作者头像（author_img）等。我喜欢最简洁的风格，因此没有加图片。
  4. 主题颜色theme_color: 可以在一些设计师网站中寻找一个最喜欢的配色方案。
  5. 作者信息author: 尽量多地写一些自己的信息。
  6. 如果你对这个主题的整体配色都不满意，可以在jacman/source/css/_base/variable.styl中修改。
    还有很多属性可以自己研究。如果有疑问可以参考这篇文章：如何使用jacman主题

杂项

图片存储

如果你上传的图片过大，github提供的空间不够了，可以用七牛云存储，注册后填一下个人信息即可领取10G的免费空间，再也不用担心内存了！

404页面

可以在github上你的空间里直接添加404.html，推荐使用腾讯公益404。

提交给搜索引擎

推荐提交给google，审核速度快，不用担心词条顺序，而且感觉google的爬虫更高效，解析网页更彻底！
链接如下： https://www.google.com/webmasters/tools/submit-url?hl=zh-CN
点击“进行人机身份检验”之后输入域名提交即可。
